# Vendor
## Administration Service
- Configuration of Base Services
>- Self Service
    - reset DB on DEV
    - create a copy of PROD on STAGE
    - email
> - Download vendorized software

# Vendor
## Customer Service

- Vendors provide licenses to customers
- Customers can be partners
- Customers have branches
- Customers have contracts & entitements
- Customers have contacts

# Vendor
## Contract Service
- Vendor create contracts about customer rights
- Contract can define
> - Subscriptions of  products
    - renewal cycle 
> - General maintenance & warranty
    - for a customer, branch, partner
    - for a product
+ the result of a contract can be an entitlement
+ contract data may end up in license containers

# Vendor
## Catalog Service
- Definition of Products
    > - License Models
    > - Features
- Definition of License Container
    > - Deployment mechanism for enforcement
- Definition of License Domains
    > - for License Container Rehost
    > - for Token Exchange

# Vendor 
## Entitlement Service
- Definition of a licensing order
- Can be generated by Contract Service
- May have a recurrence pattern (renewal)

# Vendor
## Activation Service
- Create licenses
- Revoke licenses
- Storage of licenses
